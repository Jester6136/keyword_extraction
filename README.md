# Keywords extractor

Keywords for extracting newpaper

## Description

AIA Academy project

## Getting Started

### Dependencies

* Java 1.8+ (JRE or JDK)
* Python 3.6+

### Installing

* To install this project, users have to run the following command:

    `$ pip install -r requirements.txt` 

    `$ pip install -e .` 

### Executing program

* To run demo the program:

    `$ uvicorn api.main:app --reload`
